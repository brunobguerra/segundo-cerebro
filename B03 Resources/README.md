# B03 Resources

Camada **Recursos** do PARA: conceitos estáveis, páginas por fonte e entidades.

- **`sources/`** — uma nota por fonte ingerida (com caminho para `A00 Inbox/clippings/`).
- **`concepts/`** — definições e ligações.
- **`entities/`** — pessoas, organizações, etc.
- **`tools/`** — `wiki_search.py` (pesquisa local; podes correr a partir da raiz do repositório; o default de `--root` aponta para a raiz do Git).

O agente mantém estas pastas segundo `.cursor/CLAUDE.md` (exceto scripts em `tools/`, que evolues com cuidado).
