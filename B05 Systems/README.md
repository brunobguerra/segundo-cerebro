# B05 Systems

Núcleo operacional da wiki: **índice**, **visão geral**, **log** e **meta** (governança).

- `index.md` — catálogo; o agente atualiza após ingests.
- `log.md` — linha do tempo append-only.
- `meta/` — notas como `operacao-ia` (fluxos e papéis).
- `Templates/` — `nota-padrao.md` (wiki `B*`, frontmatter alinhado a `CLAUDE.md`; variáveis `{{title}}` / `{{date:YYYY-MM-DD}}` para o plugin **Templates** do Obsidian) e `daily-note.md` (diário humano, ex.: pasta `A04 Daily Notes/` nas definições de **Daily notes**).
